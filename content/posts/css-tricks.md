Here are some CSS tricks I use regularly:

## Center Element

\`\`\`css
.parent {
  display: grid;
  place-items: center;
}
\`\`\`

## Aspect Ratio

\`\`\`css
img {
  aspect-ratio: 16 / 9;
  object-fit: cover;
}
\`\`\`

## Custom Scrollbar

\`\`\`css
::-webkit-scrollbar {
  width: 8px;
}
\`\`\`

CSS has come a long way. Use modern features!
